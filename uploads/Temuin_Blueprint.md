# Temuin Blueprint

## Branding

-   Nama: **Temuin**
-   Tema utama: **Merah** (#DC2626) dengan aksen putih, abu-abu, dan
    hitam.
-   Tagline: **Barang hilang? Temuin aja.**

## Visi

Platform AI Lost & Found khusus Indonesia untuk barang, dokumen, dan kendaraan
hewan peliharaan.

## Target

Masyarakat Indonesia, mahasiswa, pekerja, UMKM, pengguna transportasi
umum.

# User Roles

## Guest

-   Melihat landing page
-   Mencari laporan
-   Melihat detail laporan

## User

-   Login/Register
-   Membuat laporan
-   Chat internal
-   Bookmark
-   Komentar
-   Notifikasi
-   Menandai barang kembali

## Admin

-   Moderasi laporan
-   Kelola user
-   Kelola kategori
-   Dashboard statistik
-   Review AI
-   Kelola spam

# User Flow

Landing → Cari Barang → Detail → Login → Dashboard → Buat Laporan → AI
Analisis → Menunggu Verifikasi → Admin Approve → Publish → AI Smart
Match → Chat → Serah Terima → Barang Kembali → Arsip.
